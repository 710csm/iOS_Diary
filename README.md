# iOS_Diary  

## 설명   
FSCalendar 라이브러리와 Firebase를 활용한 일기 작성 애플리케이션   
FSCalendar를 활용하여 달력을 구현하고 Firebase를 통해 값을 저장하고 불러온다   
작성한 일기를 수정 및 공유 삭제할 수 있다
